99.9, The Zorg

99.9 Radio Station Infomercial

While creating this project I certainly learned a few new tricks. I had to create this project using adobe premier pro and feel as if I did a pretty good job with the requirements. The video is an infomercial for a trip to Ocho Rios, Jamaica for any lucky caller who tunes into 99.9 The Zorg.

I used 4 different video clips with an array of background sounds and music to create this project. I really hope you enjoy my creative process.
